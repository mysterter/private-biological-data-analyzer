Private Biological CSV Analyzer

Standalone local tool for validation, descriptive summaries, and a user-specified regression. Runs entirely inside this browser tab.

Privacy

This page processes your CSV entirely inside this browser tab. It contains no analytics, network calls, cloud uploads, or external libraries, and it reads no file other than the one you pick below. Closing or refreshing the page clears the loaded data from memory.

Do not place the private CSV in a GitHub repository. Keep the analyzer and the data in separate folders.

5. Project information (optional)

Governance context for the report below. Every field is optional, nothing here is checked against the data, and what you type is used only to fill in the report you download. Leave a field blank and it is recorded as (not stated).

6. Governance and reproducibility report

An aggregate record of what was analysed and how: application version, timestamp, row and column counts, column mappings, missingness, the validation rules applied and their flag counts, transformations, the analysis plan, the model, and the limitations that apply to it.

What this report never contains: raw rows, specimen identifiers, file names, or any individual value. It is built from counts and summaries only, and group summaries covering fewer than five records are withheld entirely.
