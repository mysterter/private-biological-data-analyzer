Private Biological CSV Analyzer

Standalone local tool for validation, descriptive summaries, and a user-specified regression. Runs entirely inside this browser tab.

Privacy

This page processes your CSV entirely inside this browser tab. It contains no analytics, network calls, cloud uploads, or external libraries, and it reads no file other than the one you pick below. Closing or refreshing the page clears the loaded data from memory.

Do not place the private CSV in a GitHub repository. Keep the analyzer and the data in separate folders.
