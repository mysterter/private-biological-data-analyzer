Private Biological CSV Analyzer — tests

Double-click this file to check that the tool still behaves correctly on this machine. No server, no network, no installation.

What is checked here

These are the same test cases that node tests/run-tests.js runs: CSV quoting, missing required values, duplicate identifiers, nonpositive measurements, organ masses above body mass, the GSI and HSI calculations, the regression maths, the analysis plan, the network kill switch, and the governance report — that its exports carry no specimen identifiers, that small groups are suppressed, that missingness is computed correctly, and that identical inputs produce identical output.

Two groups of checks are missing here because a page opened by double-clicking is not allowed to read files on its own: the end-to-end pass over synthetic_example.csv and the scan of the source files for network references. Both run in the command-line runner.
